## 스파르탄 메타버스 프로젝트
- 2D 탑뷰 시점의 게임으로 타일맵으로 구성된 맵에서 다른 오브젝트와 상호작용하며 NPC 대화, 미니게임 진입, 미니게임 결과에 따른 리더보드를 이용할 수 있다.
- W,A,S,D 그리고 방향키를 통한 이동을 지원하며, Z키를 통해 탈 것을 탑승할 수 있다.
- 물이 포함된 맵을 구성하여, 물에 들어갔을 때 이동 속도가 느려지며, 이펙트가 발생하도록 하였다.
